## 📌 概要

このアプリは、日々のトレーニングを記録・管理できる**筋トレ記録アプリ**です。  
カレンダー形式でログを振り返りながら、目標設定やトレーニング内容を記録できます。

**友人と共同で設計・開発**を行いました。

---

## 🚀 使用技術

- **Next.js**（App Router）
- **React**
- **TypeScript**
- **Tailwind CSS**

---

## 🔧 主な機能

- カレンダーから日付を選択し、トレーニング内容を記録
- 種目・回数・セット数・重量などの詳細入力
- トレーニング目標の設定／削除
- 登録済みの目標・記録の一覧表示
- スマートフォン対応（レスポンシブ対応済み）

---

## 📱 スクリーンショット

<img src="./スクショ1.png" alt="筋トレアプリスクリーンショット" width="600" style="margin-bottom: 20px;" />

- 🗓 カレンダー機能
月ごとのカレンダーから日付を選択し、その日のトレーニング内容を記録できます。選択された日は青くハイライト表示されます。

- 📝 トレーニング記録入力
種目名、重量、回数などの情報を入力し、「記録を追加」ボタンで保存します。未入力のフィールドは「0」と表示され、エラーを防止。

- 📊 合計負荷のグラフ表示
選択された週や月のトレーニング合計負荷（トン単位）を棒グラフで可視化。進捗やトレンドが一目で確認できます。

- 🎯 トレーニング目標登録機能
目標（例：「1ヶ月で体脂肪-3%」など）を入力し、「追加」ボタンで保存。登録された目標は下部に一覧表示されます。

- 📱 レスポンシブデザイン
モバイルでも使いやすいUI設計。画面下部にはナビゲーションバーがあり、ホーム・トレーニング・体組成の各画面にスムーズに遷移できます。

<img src="./スクショ2.png" alt="筋トレアプリスクリーンショット" width="600" style="margin-bottom: 20px;" />

- 📂 部位別分類表示
胸・背中・脚・腕・肩・腹筋など、主要な部位ごとにトレーニング種目を分類して表示。視認性が高く、目的の種目をすぐにYouTubeで見つけられます。

- 📌 代表的な種目例

    胸：ベンチプレス、ペックフライ、チェストプレス

    背中：ラットプルダウン、ベントオーバーロー、デッドリフト

    脚：スクワット、レッグプレス

    腕：アームカール、バーベルカール、ケーブルプレスダウン

    肩：サイドレイズ、ショルダープレス、フロントレイズ

    腹筋：プランク、上体起こし

- 🗑 削除ボタン付き（赤いゴミ箱アイコン）
種目ごとの削除が可能。誤って登録した内容や不要な種目を簡単に管理できます。

- 📱 下部ナビゲーションバー
他の画面（ホーム、トレーニング記録、体組成管理）への移動もスムーズに行えます。

<img src="./スクショ3.png" alt="筋トレアプリスクリーンショット" width="600" />

- 📅 日付入力
測定日をカレンダーから選択可能。毎日の体組成記録を日付ごとに管理できます。

- 🧍‍♂️ 基本情報の記録
以下の項目を入力・記録できます：

    体重 (kg)

    体脂肪率 (%)

    筋肉量 (kg)

    ウエスト (cm)

- 🧮 自動計算項目（入力に応じて自動反映）

    BMI（体格指数）

    体脂肪量

    除脂肪体重

    筋重量（※筋肉量とは別に表示）

- 📝 メモ欄付き
トレーニングの感想や体調メモなど、自由にコメントを残せます。

- ⚙️ 編集・削除機能付き
入力欄の横にあるアイコンから、個別の編集や削除も可能。

- 📱 モバイル対応デザイン
画面下部のナビゲーションバーから、「ホーム」「トレーニング」「体組成」画面にワンタップで移動可能。
